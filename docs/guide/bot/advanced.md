# AWMC BOT 指令帮助

::: tip 提示
善用搜索快捷键 <kbd>/</kbd>（电脑端）或顶部搜索按钮（手机端）进行查询。
:::

本文档分为两大块：**传输 Bot** 负责账号绑定、B50 上传、票券与卡密等数据传输；**查询 Bot** 负责曲目查询、成绩图、定数表与群互动等查分展示。

---

## 一、传输 Bot（MaiBot）

指令前缀以 `/` 为例，与 Koishi 配置一致即可。绑定账号时支持 **SGID 文本**或**公众号提供的网页地址**。

### 1.1 帮助与信息

| 命令 | 说明 |
|------|------|
| `/mai` / `/mai帮助` | 查看所有可用指令；可加 `--advanced` 展开发票、收藏品、舞里程等高级功能 |
| `/maiping` | 测试机台连接 |
| `/maiqueue [Ref_ID]` | 查询发票充值队列状态（可带 Ref_ID 查单笔） |
| `/mai状态` / `/mymai` | 查询绑定状态、票券与优先授权；可加 `--expired` 查看过期票券 |
| `/mai地图 [二维码或链接]` | 查询账号游玩过的省份/地区 |
| `/mai查询opt <版本号>` | 查询 Mai2 选项文件下载地址，如 `/mai查询opt 1.40` |

### 1.2 账号绑定

| 命令 | 说明 |
|------|------|
| `/mai绑定 [二维码或链接]` | 绑定舞萌 DX 账号；无参数时交互式引导 |
| `/mai解绑` | 解绑舞萌 DX 账号（受换绑冷却限制） |
| `/mai解绑卡` / `maiunbindkey` | 冷却期内凭**解绑卡额度**解绑（需 SGID 验证与二次确认） |

### 1.3 水鱼 B50

| 命令 | 说明 |
|------|------|
| `/mai绑定水鱼 <token>` | 绑定水鱼 Token，用于上传 B50 |
| `/mai解绑水鱼` | 解绑水鱼 Token（舞萌 DX 绑定保留） |
| `/mai上传B50` / `maiu` | 上传 B50 到水鱼 |
| `/maiua [二维码或落雪码]` | 同时上传 B50 到水鱼和落雪（SGID 只需提供一次） |

### 1.4 落雪 B50

| 命令 | 说明 |
|------|------|
| `/mai绑定落雪 <Token>` | 绑定落雪 Token（个人中心第三方导入） |
| `/mai解绑落雪` | 解绑落雪 Token（舞萌 DX 绑定保留） |
| `/mai上传落雪b50` / `maiul` | 上传 B50 到落雪 |

### 1.5 账号安全

| 命令 | 说明 |
|------|------|
| `/mai锁定` | 锁定账号，防止他人登录 |
| `/mai解锁` / `/mai逃离` / `mai逃离小黑屋` | 解锁账号（仅限通过 `/mai锁定` 锁定的账号） |
| `/mai保护模式 [on\|off]` | 开关保护模式；开启后账号下线时自动尝试锁定 |

### 1.6 优先授权与卡密

开启 **priorityCooldown** 后，部分指令会有冷却；兑换卡密可缩短或免除冷却。

| 命令 | 说明 |
|------|------|
| `/mai兑换卡密 [卡密]` | 兑换卡密；可直接带卡密，或发送指令后在限时内粘贴（一般为 `MAI-` 开头） |
| `/mai取消群组优先` | 在群内取消本群群组优先（仅**群组卡兑换人**可操作） |
| `/mai群组优先换绑` | 在**原群**发起群组优先迁移 |
| `/mai群组优先换入` / `mai群组优先换绑完成` | 在**目标群**完成群组优先迁移 |

#### 卡密类型

| 类型 | 兑换环境 | 说明 |
|------|----------|------|
| **个人卡** | 私聊或群均可 | 该绑定账号**全局**享受优先冷却 |
| **群组卡** | **必须在对应群聊内**兑换 | 绑定到当前群；群内成员在**群内**使用指令时免冷却（私聊不享受） |
| **解绑卡** | 私聊或群均可 | 须已 `/mai绑定`；增加解绑额度，用于冷却期内的 `/mai解绑卡` |

### 1.7 高级功能（票券、收藏品等）

::: danger 警告
发票有风险，操作需谨慎。AWMC TEAM 对发票可能造成的负面影响概不负责。

参见 [AWMC BOT 服务协议与隐私政策](/guide/bot/terms#十、免责声明) 第十条。
:::

| 命令 | 说明 |
|------|------|
| `/mai发票 [倍数] [目标]` | 发放功能票（倍数 2–6，默认 2）；4 倍及以上易失败 |
| `/mai修改版本号 [二维码或目标]` | 修改游戏版本号（支持缓存） |
| `/mai获取收藏品 [SGID或@用户]` | 交互式获取收藏品（姓名框、称号、头像、礼物、乐曲、解锁谱面、旅行伙伴等） |
| `/mai上传乐曲成绩 [@用户]` | 交互式上传单曲成绩 |
| `/mai删除成绩 [@用户]` | 交互式删除单曲成绩 |
