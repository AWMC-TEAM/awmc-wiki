# AWMC BOT 指令帮助

::: tip 提示
善用搜索快捷键 <kbd>/</kbd>（电脑端）或顶部搜索按钮（手机端）进行查询。
:::

账号绑定、B50 上传、票券、PC 同步、成绩查询与群互动现已合并到同一个 QueryBot。
指令前缀由平台配置决定；下文以不带前缀为例。

## 一、账号、上传与 BREAK

开始前请先发送 `用户协议`，打开网页阅读并完整复制确认词。二维码相关指令支持
`SGWCMAID...`、Wahlap `img/MAID...png` 与 `req/MAID...html` 官方链接。

### 1.1 帮助与信息

| 命令 | 说明 |
|------|------|
| `mai账号` | 查看账号功能帮助 |
| `mai状态` / `mymai` | 查看完整账号状态；二维码缓存失效时交互刷新 |
| `舞萌状态` / `mais` | 「服务器失败率」折线图（账号操作全量：发票、maiu 上传、绑定等；`returnCode=0` 计失败；无数据时段省略）+ 服务器实时状态 |
| `maiping` | 测试 AWMC API 连接 |
| `maiqueue` | 查询公共网关队列状态 |

::: tip 舞萌开字母
需先由群管发送 `开启mai猜歌`；与猜歌 / 曲绘 / 曲子 / 铺面**同群互斥**。

- 对局中直接发单个字母（如 `m`）开字符，或发曲名/别名猜中；也可 `开字母 m`、`开歌 xxx`
- 字母补齐完整标题时记入补齐者贡献（✅ 字母补齐）
- **局内不计分、不发 BREAK**；全部解开后按**用时星级 + 个人贡献**统一结算
- 用时精确到毫秒，展示为 `xx.xxx秒`
- **星级（自适应）**：默认 ≤30 / 45 / 60 / 90 / 180 秒对应 5～1 星；群内最近通关变快后，按历史用时 P35 收紧五星上限（最低 15 秒、最高仍 30 秒），其它星按 1 : 1.5 : 2 : 3 : 6 相对五星缩放；更慢为 0 星最低奖池。结算文案会显示本局生效阈值
- **贡献权重**：有效开字母 ×1、补齐曲 ×3、开歌 ×4；无贡献不得分；奖池按权重分配
- 通关结算顺序：① 猜中/补齐文案 + 用时星级 + 奖池 ② 本局榜单·贡献分成图 ③ 全开看板图；亦可随时查 `开字母排行` / `开字母贡献榜` / `开字母时间榜`
- `不玩了` / `结束开字母`：结束并揭晓剩余，**不发**速度奖与贡献奖（不用通关奖池文案）
:::

::: warning 答题限流
同一用户全局 **3 秒**只能提交一次答案（猜歌 / 曲绘 / 曲子 / 铺面 / 开字母）。超限时提示：

`嘿嘿，你的答案被我吃掉啦！(x.x秒后才能发送新的答案）`
:::

### 2.10 AWMC BREAK

| 指令 | 说明 |
|------|------|
| `AWMC签到` | 每日签到获取 BREAK |
| `我的AWMC` | 查看账号状态与使用统计（含发票 returnCode 统计） |
| `AWMC帮助` | 查看 BREAK 系统说明 |
| `转账BREAK @用户 数量` | 向其他用户转账 BREAK |
| `BREAK抽奖 [1-10]` | 默认每次消耗 2 BREAK，可一次抽 1～10 次 |

::: tip BREAK 食用说明

有修改建议请前往 [AWMC 论坛](https://bbs.wmc.pub/forums/awmc.19/) 提出。接入 AI 后会自动修改机器人并更新。

1. 每日签到可以获得 BREAK，连续签到可以获得更多 BREAK。
2. 中午、晚上的高峰期，锐评可能消耗双倍 BREAK。
3. 可以使用 BREAK 抽奖，最多支持 10 连抽。
4. 猜歌成功可以获得 1 BREAK，不设上限。
5. 「今日舞萌」会根据人品值增加 BREAK。
6. 和群主谈情说爱，可能会获得一些 BREAK。

:::

> 查分指令每日首次实际请求查分器 API 免费，之后每次扣 1 BREAK（缓存命中不扣）。
> 猜歌每次猜对默认获得 1 BREAK；成绩上传和发票只在外部操作真正成功后结算，
> 并分别享有每日首次成功免费。
