# AWMC QueryBot 使用说明

账号绑定、成绩上传、PC 同步、查分与 BREAK 互动现已合并在同一个 QueryBot 中，
无需再区分“账号 Bot”和“查询 Bot”。本文中的指令默认不写命令前缀；如果当前平台
配置了 `/` 等前缀，请在指令前补上对应前缀。

::: tip 交流与反馈
加入 QQ 群参与交流：[1072033605](https://qm.qq.com/q/7157yt6n6w)。

遇到异常时请保留 Bot 返回的 `Ref_ID`，管理员可据此查看脱敏后的完整请求链路。
:::

::: danger 凭据安全
`SGWCMAID`、舞萌官方二维码链接和查分器 Token 都属于敏感凭据。Bot 收到后会优先
尝试撤回原消息；如果提示撤回失败，请立即手动撤回，不要把凭据转发给他人。
:::

## 一、首次使用

### 1. 阅读并确认用户协议

1. 向 Bot 发送 `用户协议`。
2. 打开 Bot 返回的 [AWMC maiBot 服务协议（v4）](https://wiki.awmc.team/guide/bot/terms)。
3. 认真阅读完整网页，并复制页面中的**完整确认词**。
4. 将确认词直接发送给 Bot；也可以发送 `同意用户协议 <完整确认词>`。

确认成功后，才可使用账号绑定、上传、发票、BREAK 转账和抽奖等功能。发送
`撤回用户协议` 可以撤回同意，但不会自动删除已有账号绑定。

### 2. 绑定舞萌账号

发送：

```text
mai绑定
```

Bot 会要求提交最新的玩家二维码数据。获取方式：

1. 打开微信中的「舞萌DX | 中二节奏」玩家二维码；
2. 长按二维码，选择「识别图中二维码」；
3. 复制识别出的字符或网页地址，发送给 Bot。

支持以下三种内容：

- 完整的 `SGWCMAID...` 字符串；
- `https://wq.wahlap.net/qrcode/img/MAID....png` 图片链接；
- `https://wq.wahlap.net/qrcode/req/MAID....html` 请求链接。

也可以一次发送 `mai绑定 SGWCMAID...`。常用别名：`maibind`。

Bot 会验证二维码中的账号身份，绑定成功后只显示玩家名和 Rating，不会发送街机 UID。
如果同一个舞萌账号存在于旧 Bot 迁移记录中，新凭据验证成功后会自动认领旧记录；
BREAK、协议状态和落雪 OAuth 不随认领转移。

## 二、查询账号状态

发送 `mymai` 或 `mai状态` 可查看：

- 玩家名、Rating 及新旧版本 Rating 拆分；
- 友人对战等级、游玩次数、机台与数据版本；
- 最近登录、游玩、拼机、地区与觉醒次数；
- 账号封禁状态、BREAK 余额；
- 水鱼/落雪绑定、最近上传和有效票券摘要。

状态消息不会显示完整二维码、Token 或街机 UID。

最近一次验证成功的二维码默认缓存 10 分钟。缓存有效时，Bot 会先重新验证账号身份，
再获取最新状态；缓存过期、返回空值/500、上次使用失败或身份不一致时，Bot 会重新
询问二维码。最多输入 3 次，发送 `取消` 可直接查看已保存的缓存资料。

## 三、上传成绩到查分器

### 1. 绑定水鱼

前往 [水鱼查分器](https://maimai.diving-fish.com/) 获取导入 Token，然后发送：

```text
mai绑定水鱼 <Token>
```

别名：`maibindfish <Token>`。

### 2. 绑定落雪

推荐直接发送：

```text
lxbind
```

按提示完成落雪 OAuth 授权，之后无需向 Bot 提交落雪导入 Token。

如需使用兼容方式，也可以发送：

```text
mai绑定落雪 <导入Token>
```

别名：`maibindlx <导入Token>`。上传时会优先使用 OAuth，只有未授权或需要兼容回退时
才使用导入 Token。

### 3. 执行上传

| 指令 | 作用 | 默认价格 |
|---|---|---:|
| `maiu` | 上传水鱼 | 2 BREAK |
| `maiul` | 上传落雪 | 2 BREAK |
| `maiua` | 同时上传水鱼和落雪 | 3 BREAK |

三种方式共享“每日首次上传成功免费”额度。只有外部服务真正返回成功后才结算；二维码
无效、HTTP 500、返回全空、任务失败或超时均不扣 BREAK。

上传会优先使用 10 分钟内已验证的二维码缓存。缓存不可用时，Bot 会要求发送最新的
`SGWCMAID` 或官方链接，最多重试 3 次；发送 `取消` 可结束本轮上传。

::: info 第三方查分说明
QueryBot 无法自动知道玩家何时在机台游玩、打了什么歌。机台游玩后，需要重新执行
`maiu`、`maiul` 或 `maiua`，查分器中的 B50 才会更新。
:::

## 四、直接发送二维码

直接发送以 `SGWCMAID` 或上述 Wahlap 官方链接开头的消息时，Bot 会按以下顺序处理：

1. 优先撤回原凭据消息；
2. 验证用户协议和群功能开关；
3. 获取并保存最新 PC 数据；
4. 根据绑定情况自动上传：只绑定水鱼就传水鱼，只绑定落雪就传落雪，两边都绑定就
   同时上传；
5. 合并返回 PC 数量、上传结果、BREAK 结算和 `Ref_ID`。

如果没有绑定任何查分器，本次只同步 PC。二维码无效、过期或服务暂时不可用时，
3 分钟内最多可重新发送 3 次。

## 五、PC 数据与查分

| 指令 | 说明 |
|---|---|
| `更新pc数` | 使用有效缓存或最新二维码同步 PC 数据 |
| `我的pc数` | 查看总 PC 与游玩最多的 15 个谱面 |
| `pc数 <歌曲>` | 查看指定歌曲的 PC 数据 |
| `pc50` | B50 按游玩次数排序 |
| `pca50` | Rating B50 内按 PC 重新排序 |
| `游玩排行50` | 全库游玩最多的 50 个谱面 |
| `pc排行` | 全部已同步用户的 PC 总量排行 |

PC 数据来自舞萌账号同步；`b50`、`ab50` 等成绩图仍以水鱼或落雪查分器为准。

如果刚完成上传但查询仍显示旧成绩，可以稍等片刻后发送 `刷新b50`。Bot 会在上传前后
主动清理本地成绩缓存，但查分器本身也可能需要时间处理。

## 六、BREAK

### 1. 获取与查看

| 指令 | 说明 |
|---|---|
| `AWMC签到` / `签到` | 每日签到获取 BREAK |
| `我的AWMC` | 查看余额、连续签到、今日使用量和最近流水 |
| `AWMC帮助` | 查看 BREAK 规则 |

签到基础奖励默认为 1～2 BREAK，连续签到最高额外增加 2 BREAK。指定群、周四和群内
当日首签可能获得额外倍率，实际结果以 Bot 返回为准。

### 2. 默认消费规则

| 业务 | 默认规则 |
|---|---|
| 查分器 API | 每日首次实际请求免费，之后 1 BREAK；缓存命中不扣 |
| B50 分析 | 3 BREAK，高峰时段可能按规则加倍 |
| 猜歌 | 每次猜对固定获得 1 BREAK，不设每日上限 |
| 上传成绩 | 每日首次成功免费，之后水鱼 2、落雪 2、同时上传 3 |
| 发票 | 每日首次成功免费，之后为倍率 × 3 BREAK |
| 抽奖 | 默认每次消耗 2 BREAK，可一次抽 1～10 次 |

### 3. 群互动

```text
转账BREAK @用户 10
BREAK抽奖
BREAK抽奖 5
```

`BREAK抽奖` 默认抽 1 次，数字表示连抽次数。BREAK 仅为 Bot 内部互动积分，不具有
现金价值。

## 七、发票

发送 `发票 2` 或 `fp 2` 请求 2 倍票，当前支持 2、3、5 倍：

| 倍率 | 每日首免后价格 |
|---:|---:|
| 2 倍 | 6 BREAK |
| 3 倍 | 9 BREAK |
| 5 倍 | 15 BREAK |

只有发票真正成功并确认到账时才扣费。使用 `mai查票` 或 `查票` 可按票种、库存和
有效期查看票券状态；回复不会显示舞萌账号 UID 或原始接口数据。

::: danger 风险提示
发票、上传及二维码相关功能可能影响舞萌官方账号安全。使用前请完整阅读
[AWMC BOT 服务协议与隐私政策](/guide/bot/terms)，并自行承担相应风险。
:::

## 八、常见问题

### Bot 提示“尚未建立账号记录”

依次完成：

1. 发送 `mai绑定`，再提交最新的 `SGWCMAID` 字符串或官方链接；
2. 按需发送 `mai绑定水鱼 <Token>`，或使用 `lxbind` 绑定落雪 OAuth；
3. 使用 `maiu`、`maiul`、`maiua` 上传水鱼、落雪或两边。

### 二维码返回 500、全是 null 或提示过期

返回舞萌玩家二维码页面重新生成并长按识别。不要重复发送旧二维码；同一轮流程最多
允许 3 次，仍失败时请稍后重新开始。

### Bot 无法撤回凭据消息

请立即手动撤回。Bot 仍会继续返回脱敏后的处理结果，但无法替代用户删除原消息。

### 查分提示用户不存在或没有数据

确认水鱼个人资料填写了正确的 QQ 号，并关闭“禁止其他人查询我的成绩”。首次绑定后
至少执行一次 `maiu`；使用落雪时请检查 OAuth 是否仍有效。

### B50 成绩是整数或与网页不一致

检查水鱼隐私设置，关闭“对非网页查询的成绩使用掩码”，然后发送 `刷新b50` 重试。

### 上传提示登录失败

通常是二维码已过期、账号仍登录在机台，或外部服务暂时不可用。请重新获取二维码，
按照 Bot 提示最多重试 3 次；失败流程不会扣 BREAK。
