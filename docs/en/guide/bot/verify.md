# Authorization
This Bot's maimai features require authorization. Authorization scope is per group.

:::danger Unauthorized Notice
If you add the bot to a group without prior approval, the bot will not provide maimai-related services.
:::

## Free Arcade Authorization

Official arcade player groups (with an official designation) can apply for free authorization:
![](/images/official_arcade_example.png)

Please [send us an email](mailto:awmc@awmc.cc) with the following information:

> Subject: Group number + "Arcade Authorization Request"
> 
> In the body, include the arcade address + a screenshot of the group info page.
